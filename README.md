Project Overview

This project is created to learn and practice Functions in ReactJS, especially how event handling works using onClick, mouse events, and calling functions inside other functions.

📚 Topics Covered

Functional Components

Event Listening in React

onClick Event

Mouse Events (onMouseEnter, onMouseLeave)

Calling Function Inside Another Function
